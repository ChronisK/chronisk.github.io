Save / Upload Configuration

Save

In order to save the current map configuration (colors of countries-states, borders, map title and labels), click the button below to download a small mapchartSAVE.txt file containing all the information.

Save map configuration

Upload

When you are ready to continue your work, copy and paste below the content of the mapchartSAVE.txt file you previously downloaded. WARNING! The current map will be cleared of all data.

-

United Kingdom (& Ireland)

  • England's 48 Ceremonial Counties
  • Scotland's 35 Lieutenancy Areas
  • Wales' 8 Preserved Counties
  • Ireland's 26 and N. Ireland's 6 Counties
Step 1

Single Selection

Select the color you want and click on any county on the map. You can right-click on a county to remove its color.

More Options

Search for a Subdivision and Color it

Search for the subdivision you want and click COLOR SUBDIVISION to fill it with the current Fill Color. To remove the color of the selected subdivision, choose REMOVE COLOR .

Step 2

Add the title you want for the chart's legend and choose a label for each color. Change the color for all counties in a group by clicking on it.

Color Label Remove
Step 3

When ready, select Preview Map . To download the generated Chart: Right Click -> Save Image or select Download Map . In case you want to change something, select Edit . To start over, select Clear All .
